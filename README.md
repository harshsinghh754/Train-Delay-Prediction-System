# 🚆 Train Delay Prediction System

This project is a Machine Learning-based web application that predicts train delays using historical train data, weather conditions, and station-specific patterns. It helps users understand and anticipate delays through predictive models and visual insights.

## 🔍 Features
- 📊 Predict train delays using regression and classification models
- 🌦️ Incorporates weather data for better accuracy
- 🗺️ Interactive heatmap visualization of delays
- 📁 Clean Flask-based backend structure
- ⚡ Fast and user-friendly interface

## 🛠️ Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas, NumPy
- HTML, CSS (Jinja Templates)

## 📂 Project Structure
- `app.py` → Main Flask application
- `train_ml_model.py` → Model training script
- `generate_map.py` → Heatmap generation
- `templates/` → Frontend HTML files
- `.pkl files` → Trained ML models
- `.csv files` → Dataset

## 🤖 Models Used
- Delay Classification Model
- Delay Regression Model

## 📈 Use Case
This system can be used by passengers, railway analysts, or developers to:
- Predict delays in advance
- Analyze station-level delay trends
- Improve travel planning

## 🚀 Future Improvements
- Real-time train API integration
